## Features

- Pop up Menu
- App Navigation


## Requirements
- Xcode 9.0+
- Swift 4.0+
- iOS 8.0+


## Installation
You will need CocoaPods to use the EasyNav

In order to Integrate EasyNav to your XCode Project , specify it in your PodFile

pod 'EasyNav','~> 1.0.1'

then in your code add 'Import EasyNav'
